How To Use?

The sending form accepts data about the anthill.

The data about the anthill consists of 3 blocks.
* Block 1 is the number of ants
* Block 2 is information about rooms
* Block 3 is the relationship between rooms

Rules for [block 1] the number of Ants must be greater than zero
Rules for [block 2] the room must consist of a name, X coordinates and Y coordinates separated by a space (Name X Y). Each room in 1 line
Rules for [block 3] the rooms are connected to each other by a hyphen. Each relation is in 1 line
and all this in strictly in sequence [block 1, block 2, block 3] (These rules apply to [Text Result] too)

After filling in, click on Send and you will get the result in Graphic and text form. The graphic result depends on the Text result. The text result is converted to graphic via the Set Params button. In the graphical result, each path will have its own color, unused paths will be gray. To apply any changes in the text result or in the parameters then press Set Params.

Also you can use web version for visualize your result. To do this, insert the map with result into the `text result` block and click on Set Params. Now your graph is represented graphically.
